# 项目简介

## 核心功能
- 数据采集
- 数据预处理
- 数据分析
- 模型建立：决策树、随机森林、SVC、线性回归、KNN、高斯朴素贝叶斯、梯度升级、XGBoost、梯度提升
- 模型优化
- 数据可视化

## 适用场景
- 电信行业用户流失分析
- 客户行为预测
- 客户价值评估

## 技术特点
- 使用pandas、numpy、matplotlib等Python库进行数据分析和可视化
- 支持多种机器学习算法，满足不同场景的需求
- 提供源码、报告及数据集，方便用户自行运行和扩展
